    0.1. はじめに¶

    この演習資料は大学での講義用で，Image Processing（画像処理）とComputer Graphics（コンピュータグラフィックス）の入門向けのオンライン学習資料です．

    Image Processing や Computer Graphics 関連の技術に触れるため，Python を使ってそれぞれの基本技術に触れることを目的としています．

    1.1. プログラミング環境について¶

    プログラミング言語は Python，プログラミング環境は Web 上で Python を実行できる Google Colaboratory (略称: Colab）を利用します．

    Colab を利用することでブラウザから特に環境構築なしに以下の機能を使えます．

    • 必要なライブラリが揃った状態ですぐプログラミングを始められる
    • ノートブック形式の分かりやすいプログラミング環境
    • ファイルを簡単に Drag&Drop でアップロード／ダウンロード
    • Google Drive との連携

    以下では，Colab 上で本資料の Python コードを実行するために必要なもの，演習準備について触れていきます．

    1.2. 必要なもの¶

    • Google アカウント: Colab の動作，Google ドライブとの連携に必要になります．
    • Web ブラウザ: Google Chrome 推奨（筆者の実験環境）ですが，ほとんどの主要ブラウザで動作します．
