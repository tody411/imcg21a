    0.2. 利用する主なライブラリ¶

    この演習で使う主なライブラリは以下の 3 つになります．

    • NumPy: 数値計算用ライブラリ
    • OpenCV: 画像処理用ライブラリ
    • Maplotlib: 図表の出力

    上記のライブラリは Python の中でもよく使われるライブラリなので， Colab 上でもデフォルトでインストールされています．

    画像表示の最小サンプル¶

    ここでは，Image Processing の演習で良く出てくる 画像表示に関するサンプルコードを なるべく最小の形で簡潔に紹介します．

    ランダム画像の表示¶

    ここでは，NumPy と Matplotlib を使ってランダム画像を表示する最小コードを載せています．

     1

    import numpy as np
import matplotlib.pyplot as plt

# ランダム画像の生成
I_random = np.random.rand(8, 8, 3)

# Matplotでの表示処理
plt.imshow(I_random)
plt.axis('off');
plt.title("Random Image");

    ここでは，NumPy の乱数生成機能を使って 8x8 サイズの RGB 画像 I を生成して表示しています．

    import cv2
import matplotlib.pyplot as plt


# 画像ファイルの読み込み
image_file = "slides.png"
I_file = cv2.imread(image_file)
I_file = cv2.cvtColor(I_file, cv2.COLOR_BGR2RGB)

plt.axis('off');
plt.title("File Image");

    画像の読み込みは，様々な方法がありますが，このチュートリアルでは cv2.imread を使います． cv2.cvtColor 等読み込みの詳細については，演習部分で確認するようにしてください．

    import numpy as np
import cv2
import matplotlib.pyplot as plt

# ランダム画像の生成
I_random = np.random.rand(8, 8, 3)

# 画像ファイルの読み込み
image_file = "slides.png"
I_file = cv2.imread(image_file)
I_file = cv2.cvtColor(I_file, cv2.COLOR_BGR2RGB)

# Matplotでの表示処理

## 1行2列の1番目: ランダム画像を表示
plt.subplot(1, 2, 1)
plt.imshow(I_random)
plt.axis('off');
plt.title("Random Image");

## 1行2列の2番目: ファイル画像を表示
plt.subplot(1, 2, 2)
plt.imshow(I_file)
plt.axis('off');
plt.title("File Image");

    NumPy で使う主な機能¶

    行列・配列の初期化

    この辺りは良く使います．

    • np.zeros: 0 で初期化
    • np.ones: 1 で初期化
    • np.random.rand: 0 ～ 1 のランダム要素で初期化

    また，画像を作成する際は，(h, w, cs)の順番になります．

    • h: 画像の高さ
    • w: 画像の幅
    • cs: チャンネル数
    1
2
    import numpy as np
I = np.zeros((90, 160, 3))

    上記のコードは，幅: 160, 高さ: 90, チャンネル数: 3(RGB 等標準)の画像を作っていることに相当します．

    Note

    幅，高さ，チャンネル数ではないので注意しましょう．

    OpenCV で使う主な機能¶

    実際の使い方については演習用の Colab ノートブックを見た方が分かりやすいと思います．

    以下の関数は，特によく使うので Web 等を参照して色々な使い方を調べてみると良いです．

    • cv2.imread: 画像の読み込み
    • cv2.cvtColor: 色空間の変換
    • cv2.GaussianBlur, ...: ガウシアンフィルタなどの各フィルタ関数
