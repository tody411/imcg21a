    プログラミング環境について¶

    プログラミング言語は Python，プログラミング環境は Web 上で Python を実行できる Google Colaboratory を利用します．

    Google Colab を利用することで

    • 必要なライブラリが揃った状態ですぐプログラミングを始められる
    • ノートブック形式の分かりやすいプログラミング環境
    • ファイルを簡単にアップロード／ダウンロード

    利用する主なライブラリ¶

    • NumPy: 数値計算用ライブラリ
    • OpenCV: 画像処理用ライブラリ
    • Maplotlib: 図表の出力

    上記のライブラリは Python の中でもよく使うライブラリで，Web 上にいろんな参考文献があるのでぜひ遊んでみてください．
